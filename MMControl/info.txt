Утилита для управления кодеками. Возможности: Отображает список кодеков по категориям, информацию о них, страницы свойств; удаляет кодеки, регистрирует новые. Есть автоматический поиск некорректно установленных кодеков.

Утилита для управления кодеками в Windows. Поддерживает кодеки в виде DirectShow-фильтров и Direct-X Media Objects. Программа позволяет выполнить следующие действия: 
просмотреть список кодеков из выбранной категории (Audio Compressors, DirectShow Filters, Video Compressors и другие); 
вывести информацию по выбранному кодеку: имя фильтра, имя СОМ-объекта, путь к файлу, GUID. приоритет; 
открыть страницу свойств кодека для настройки его параметров; 
удалить кодек из реестра;
зарегистрировать в системе новый кодек (файл DLL или AX)
изменить приоритет (merit) - для DirectShow-фильтров
зарегистрировать системные фильтры, если их настройки были повреждены
Есть возможность автоматического поиска некорректно установленных кодеков, чтобы удалить из реестра неиспользуемые записи и ускорить работу системы.
Использование программы: выберите категорию в раскрывающемся списке, выберите кодек из списка, нажмите кнопку "Управление кодеком". В открывщемся диалоговом окне отображаются данные кодека, можно проверить корректность установки кодека и удалить кодек. Для изменения приоритета кодека можно ввести его числовое значение, а можно использовать предопределенные константы.
